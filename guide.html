MY GUIDE TO TAKING OVER THE INTERNET MAKING YOUR OWN WEBSITE!

    Requirements:

    ❖ A Computer/Laptop

    ❖ Your HTML/CSS

    You need something to display! There are guides on HTML, CSS and JS(if nessecary) all over the internet.

    ❖ A domain (can be bought for as cheap as a dollar; I got mine for 6$ on Namecheap for a year).

    This is an optional step if you wanted something custom, but Github Pages hosts with a 'USERNAME.github.io' domain if you just wanted to host the website.

    ❖ Visual Studio Code

    Just makes it a lot easier to code, and is nessecary for updating website.

    ❖ A Github Account
    ❖ Github on Your Computer/Laptop

    Github, and specifically Github Pages will work as the domain host.

    ❖ Set up your code, preferably within VSC. Name your primary file 'index'(.html).

    ❖ Set up your Github account.

    ❖ Follow the steps listed here (set up a Github repository with your Github username (USERNAME.github.io)) within the Github desktop application, and add your HTML files to the folder under that name.

    Push origin

    And you have your website! Open your 'USERNAME.github.io' in your browser.
